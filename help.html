Help & Tutorials

Searching

  • Our search function is not case sensitive and ignores diacritics.
  • Our search covers all the description fields for our resources.
  • Each search term separated by a space is treated as an independent search term and linked with the AND operator, which means that all terms must appear somewhere in the description fields for a resource to be returned in the results.
  • Searches are accessible via the URL.
  • You can export your search result data in a JSON file.
