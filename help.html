Help

  • Searches are not case sensitive and ignore diacritics.
  • Search terms separated by a space are treated as independent search terms and are linked with the AND operator.
  • Truncation, wildcards, and quotation marks are not supported.
  • Searches cover all resource description fields.
  • All search terms must appear somewhere in the resource description fields for a resource to be returned in the results.
  • There are two ways to save a search: Copy the url from the results page or use the export button to download the results data in a JSON file.
  • Where can I find free online visual images about Cuba in the 50s and 60s?
  • Try searching: Cuba images 1950
  • Where can I find the most comprehensive open access data projects and resources on slavery?
  • Try searching: slavery data
  • Where can I find freely available online encyclopedias, dictionaries, etc. to support my reference work?
  • Try searching: reference
  • Where can I find a list of oral histories from Latin America?
  • Try searching: oral history
  • Where can I find the most comprehensive resources focusing on Latin America ephemera?
  • Try searching: ephemera
  • Resources with full content.
  • Resources with valuable but partial online digital content. Example: Archive of Hispanic Literature on Tape
  • Resources that are part of a larger project but include a distinctive stable link to content within our scope. Example: World Digital Library : Latin America and the Caribbean
  • Resources that include some content of our interest such as international organizations, governmental agencies, think tanks etc. but they do not provide a distinctive stable link within our scope.
  • Free does not always mean open access.
